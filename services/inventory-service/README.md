# Inventory Service

Inventory management service for Microservices Tolerant

## Overview

The Inventory Service:
-   Manages product inventory levels
-   Checks product availability
-   Reserves inventory for orders
-   Supports chaos engineering for failure injection
-   Prevents race conditions with pessimistic locking
-   Records all inventory operations and reservations
